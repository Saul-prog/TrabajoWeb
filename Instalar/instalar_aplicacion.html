Instalar aplicación

Entorno

La aplicación utiliza el entorno de Apache: XAMPP.

Para instalarlo, puede descargar el archivo pinchando en el siquiente enlace: https://www.apachefriends.org/es/download.html

Base de datos

La aplicación utiliza una base de datos con servidor en phpMyAdmin. Para instalarla, se debe descargar el archivo .sql adjunto en esta carpeta, entrar en http://localhost/phpmyadmin/, darla a "Nueva" y llamar a la base de datos "fmcia", darle a "Importar" , seleccionar el archivo fmcia.sql y "Continuar" http://localhost/phpmyadmin/
