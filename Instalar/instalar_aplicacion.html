Instalar aplicación

Entorno

La aplicación utiliza el entorno de Apache: XAMPP.

Para instalarlo, puede descargar el archivo pinchando en el siguiente enlace: https://www.apachefriends.org/es/download.html

Después, debe meter la carpeta contenedora: "Bloque2_7" descomprimida en ../xamp/htdocs/carpeta, siendo "carpeta" una carpeta creada anteriormente.

Ya tenemos el entorno instalado. Inícielo

Base de datos

La aplicación utiliza una base de datos con servidor en phpMyAdmin.

Para instalarla, debe descargar el archivo .sql adjunto en esta carpeta.

A continuación, entrar en http://localhost/phpmyadmin/, clicar en "Nueva" y llamar a la base de datos "fmcia".

Para rellenarla, debe darle a "Importar" y seleccionar el archivo fmcia.sql y "Continuar".

Con la base de datos ya actualizada, ya se puede usar la aplicación.

http://localhost/phpmyadmin/
